## 🚗 [EV-Safer](https://github.com/stopcircle/Project_EV-Safer)
> **전기차 화재 안전 알림 앱**
>
> *"지하 주차장 전기차 화재, 골든타임을 지키다."*
> * **핵심 기능**: 위치 기반 실시간 화재 알림, 배터리 고온 징후 포착(0.5초 이내), 재난 안전 커뮤니티
> * **기술 스택**: Flutter, Firebase, Naver Map API
> * 전기차 화재 시 사용자에게 신속한 대피를 돕고 골든타임을 확보하기 위한 필수 안전 앱입니다.

<br>

## 🌐 [WSCD](https://github.com/stopcircle/Project_WSCD)
> **웹 기반 실시간 메타버스 플랫폼**
>
> *"공간의 제약을 넘어, 언제 어디서나 소통하다."*
> * **핵심 기능**: 별도 설치 없는 웹 접속, PixiJS 기반 2D 렌더링, WebRTC 화상 채팅, 실시간 위치 동기화
> * **기술 스택**: Python(Flask), JavaScript, PixiJS, Socket.IO, WebRTC
> * 기존 화상 수업의 한계를 넘어, 2D 아바타로 상호작용하며 몰입감을 높이는 교육용 메타버스 플랫폼입니다.
